# Cyber Safety for University Students

An educational website designed to teach university students about common cybersecurity threats and how to protect themselves online.
Web site link ; https://cyber-safety-website.vercel.app/


## About

This project was built as part of the **GDI2100** unit at Edith Cowan University. It covers key cybersecurity topics relevant to students, including phishing attacks, password security, public Wi-Fi risks, and social engineering.

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Home page with threat overview, statistics, and embedded video |
| `phishing.html` | How to identify and avoid phishing attacks |
| `passwords.html` | Password security best practices and tools |
| `wifi.html` | Risks of public Wi-Fi and how to stay safe |
| `social-engineering.html` | Recognising social engineering tactics |
| `resources.html` | Curated collection of external cybersecurity resources |
| `contact.html` | Contact form for enquiries |

## Features

- **Responsive Design** - Mobile-first layout with breakpoints at 480px, 768px, and 1024px
- **Accessibility** - WCAG 2.1 AA compliant with skip links, semantic HTML5, ARIA labels, and keyboard navigation support
- **Modern CSS** - Custom properties, CSS Grid, animations with `prefers-reduced-motion` respect, glassmorphism effects, and gradient theming
- **No JavaScript Dependencies** - Pure HTML and CSS implementation
- **Print Styles** - Optimised print stylesheet that strips decorative elements and displays link URLs
- **Embedded Media** - YouTube video integration and responsive image handling

## Tech Stack

- HTML5 (semantic markup)
- CSS3 (single stylesheet: `styles.css`)
- No frameworks or libraries
